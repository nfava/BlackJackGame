Single player Console Blckjack game 

User has a starting balance of 100 dollars that is managed. game will end if value = 0!
use must bet a min of $5 and a max of their total bankroll

The user can stand, hit and split just like a noraml blackjack game
The goal of the user is to get as close to 21 without going over all while beating the dealer

The dealer is known as the "House" in Blackjack
